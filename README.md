# Verdant Hollow

A small Hollow-Knight-inspired action platformer built in Godot.

## Overview

Virdant Hollow follows Jin Kageyama, a wandering swordsman exploring the ruins of an ancient sanctuary reclaimed by nature.

The project focuses on creating responsive movement, satisfying combat, and atmospheric level design inspired by classic exploration platformers.

## Current Features

- Responsive character movement
- Coyote time and jump buffering
- Directional sword attacks
- Dash movement
- Attack visual effects
- Knockback system
- Animated player controller
- Component-based health and damage system

## Planned Features

- Interactive enemies
- Environmental storytelling
- Boss encounter
- Camera polish
- Combat effects and hit feedback
- Fully playable level

## Built With

- Godot Engine
- GDScript
- Custom art

## Development Goals

This project is being developed as a focused portfolio piece with an emphasis on:

- Clean architecture
- Game feel
- Animation/Art
- Combat feedback
- Level atmosphere

## Current Status

Player controller and core combat systems are implemented.
Enemy development and world building are currently in progress.
